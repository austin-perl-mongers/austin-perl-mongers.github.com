Our Declaration

  • Declares one or more variables to be valid globals within the enclosing block, file, or eval.
  • Has the same rules as a my declaration for determining visibility, but variable is not private.
  • Hide variable from use strict 'vars'.

our 
    our TYPE EXPR : ATTRIBUTES
    our EXPR : ATTRIBUTES
    our TYPE EXPR
    our EXPR

Examples 

    use strict;  # Strict for entire file

    package Foo;
    our $bar;    # $bar is $Foo::bar for rest of lexical scope
    $bar = 582;

    package Bar;
    print $bar;  # prints 582, just as if "our" had been "my"


582
